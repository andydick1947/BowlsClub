ReadMe for Member Page

1. The Member page provides information about the Membership contact details.

2. The page consists of a button for each member, listed in alphabetical order.

3.Selecting a member provides contact details for that particular member.

4. On selection, the information appears in the form of a popup. The screen behind fades.

5. Once finished, the popup has to be closed to allow the user to move on.

6. The navigation bar provides access to the "Home" page
